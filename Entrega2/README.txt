a. ¿Quién invento el sistema de control de versión de Git y por qué?
	R: Linus Torvalds inventó el sistema de control de versión de Git. Él la creo porque ya existian otros sistemas de control de versiones pero no cumplian completamente, por eso decidió que crearía una herrmienta mucho mejor.

b. ¿A quién pertenece actualmente Github y por qué?
	R: Github le pertenece actualmente a Microsoft Corporation, la adquisición se dio porque querían fortalecer su presencia en el ámbito del desarrollo web 

c. ¿Hay otra forma que no sea la terminal para trabajar con Github?
	R: Si, con la versión web, de desktop y los IDE.
